# Gmail URL Paste

A browser extension that automatically converts selected text to hyperlinks when pasting URLs in Gmail and Google Calendar

## Setup

```bash
pnpm install
```

## Development Scripts

```bash
# Chrome/Edge
pnpm dev

# Firefox
pnpm dev:firefox
```

## Build

```bash
# Chrome/Edge
pnpm build

# Firefox
pnpm build:firefox
```

## Package

```bash
# Chrome/Edge
pnpm zip

# Firefox
pnpm zip:firefox
```
